Html summary Tag

  • The summary tag in HTML is used to define a summary for the details element.
    1. The summary element is used along with the details element and provides a summary visible to the user
    2. When the summary is clicked by the user, the content placed inside the details element becomes visible which was previously hidden.
    3. The summary tag was added in HTML 5.
    4. The summary tag requires both starting and ending tag.
  • Note: The summary element should be the first child element of the details element.
